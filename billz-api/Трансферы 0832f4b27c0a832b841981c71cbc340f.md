### Список

Входящие параметры для фильтрации:

- limit - макс товаров в странице
- page - номер страницы
- departure_shops — Магазины отправки
- arrival_shops — Магазин получения
- start_date — начало периода отправки
- end_date — окончание периода отправки
- arrival_date_start — начало периода принятия трансфера
- arrival_date_end — окончание  периода принятия трансфера
- status_ids — список статусов трансферов

Тело запроса

```bash
curl -X 'GET' \
  'https://api-admin.billz.ai/v2/transfer?start_date=2024-11-18%2000%3A00%3A00&end_date=&arrival_date_start=2024-11-13&arrival_date_end=&departure_shops=6f244db4-75d6-4fac-9841-14867d45e36a&arrival_shops=70919136-9481-4c7e-8581-685d750d67a9&limit=10&page=1&status_ids=65af5e85-252c-43be-8364-8c593c87c9e8' \
  -H 'accept: application/json' \
	-H 'Authorization: Bearer ...'
```

### Добавить товар

Входящие параметры:

- Магазин отправки (`departure_shop_id`)
- Магазин получения (`arrival_shop_id`)
- ИД продукта (`product_id`)
- Кол-во товаров к перемещению (`transfer_measurement_value`) - конечное колво, при повторной отправке значение будет перезаписано

	-H 'Billz-Response-Channel: HTTP' \

### Изменить трансфер

Обновление информации по трансферу стоит использовать только для редактирования названия. Менять магазин отправления и получения выполнять, только если вы понимаете последствия

Тело запроса

```bash
curl -X 'PUT' \
  'https://api-admin.billz.ai/v2/transfer/update/:tranfer_id' \
  -H 'accept: application/json' \
  -H 'Content-Type: application/json' \
	-H 'Billz-Response-Channel: HTTP' \
	-H 'Authorization: Bearer ..' \
	-d '{
    "departure_shop_id": "70919136-9481-4c7e-8581-685d750d67a9",
    "arrival_shop_id": "6f244db4-75d6-4fac-9841-14867d45e36a",
    "name": "Новое имя трансфера"
}'
```
