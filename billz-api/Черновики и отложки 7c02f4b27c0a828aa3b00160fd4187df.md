# Черновики и отложки

### Удаление товара

Так как отложка фиксирует товары, то прежде чем удалить товар, необходимо перевести отложку в черновик, то есть отменить ее.

Продукт можно удалить, передав 0 в поле **measurement_value** 

<aside>
⚠️

**Метод**: PUT v1/order-item/:id

</aside>

**JSON параметры**

| **параметр** | **тип** | **описание** |
| --- | --- | --- |
| product_id | string | UUID продукта, который нужно удалить |
| measurement_value | number | новое количество. 0 для удаления |
| customer_id | uuid | Передается UUID покупателя, если привязан к продаже |
| use_wholesale_price | boolean | признак оптовой цены. по умолчанию false |
| sellers[].seller_id | array | массив с UUID продавцоd, к которому относится товар |

**пример JSON**

```bash
{
    "customer_id": "622c8993-9f82-45ae-a69d-63103d8506d5",
    "measurement_value": 0,
    "product_id": "31a4c573-6a24-499e-9201-fab51d343d63",
    "sellers": [
        {
            "seller_id": "90219ac7-88b7-43f6-bed9-5c6761aefa71"
        }
    ],
    "use_wholesale_price": false
}
```

**пример вызова curl**

```bash
curl  -X 'PUT' \
'https://api-admin.billz.ai/v1/order-item/571caab5-20ad-4bfa-bdcc-5c1ae785d3e1' \
  --data-raw '{"customer_id":"622c8993-9f82-45ae-a69d-63103d8506d5","measurement_value":0,"product_id":"31a4c573-6a24-499e-9201-fab51d343d63","sellers":[{"seller_id":"90219ac7-88b7-43f6-bed9-5c6761aefa71"}],"use_wholesale_price":false}'
```

### Удаление черновика

Если необходимо удалить черновик, то можно воспользоваться специальным методом

<aside>
⚠️

**Метод**: DELETE v2/order/:id

</aside>

**пример вызова curl**

```bash
curl 'https://api-admin.billz.ai/v2/order/eed7e505-df8d-43e9-a6d5-34e497bd8632' \
  -X 'DELETE' \
  -H 'Authorization: Bearer ..' 
```

В ответ приходит ID принятого сообщения, который можно игнорировать

### Отмена отложки

Отмена отложки приводит к переводу в черновик. Если потребуется удалить отложку, то это выполянется методом “Удаление черновика”

После отмены отложки товары  переведутся в активный статус и станут доступны для продажи

<aside>
⚠️

**Метод**: POST v2/order/return-postpone

</aside>

**JSON параметры**

| **параметр** | **тип** | **описание** |
| --- | --- | --- |
| order_id | string | UUID черновика, который нужно удалить |

**пример JSON**

```json
{
    "order_id": "e4c86736-d680-4dd2-b88a-84901798076e"
}
```

**пример вызова curl**

```bash
curl 'https://api-admin.billz.ai/v2/order/return-postpone' \
  -X 'POST' \
  -H 'Authorization: Bearer ..' \
  --data-raw '{"order_id":"e4c86736-d680-4dd2-b88a-84901798076e"}'
```

В ответ приходит ID принятого сообщения, который можно игнорировать
