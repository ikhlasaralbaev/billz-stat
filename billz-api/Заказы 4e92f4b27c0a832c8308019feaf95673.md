### Создание нового заказа

Метод создаёт новый заказ. Товары включаются в него отдельным методом

**параметры**

| **Поле** | **Описание** |
| --- | --- |
| name | Название заказа |
| accepting_date | Дата принятия заказа |
| is_from_file	 | Индикатор загрузки из файла |
| payment_date	 | Дата оплаты |
| shop_id	 | Уникальный идентификатор магазина |
| supplier_id	 | Уникальный идентификатор поставщика |

**пример вызова**

```bash
curl 'https://api-admin.billz.ai/v2/supplier-order' \
  -X 'POST' \
  -H 'Authorization: Bearer ...' \
  -H 'Content-Type: application/json' \
  -H 'Billz-Response-Channel: HTTP' \
  --data-raw '
  {
    "name": "новый заказ",
    "accepting_date": "2024-12-17",
    "is_from_file": false,
    "payment_date": "",
    "shop_id": "6f244db4-75d6-4fac-9841-14867d45e36a",
    "supplier_id": "28110831-9bc5-485f-830c-18472f9481b1"
}'
```

### Добавление товаров

после того как заказ создан,  в него можно добавить товары

**параметры**

| **Поле** | **Описание** |
| --- | --- |
| ordered_measurement_value | Заказанное количество товара |
| product_id | Уникальный идентификатор товара |
| retail_price | Розничная цена товара |
| supplier_order_id | Уникальный идентификатор заказа поставщика |
| supply_price	 | Закупочная цена товара |

**пример вызова**

```bash
curl 'https://api-admin.billz.ai/v2/supplier-order/add-item' \
  -X 'PUT' \
  -H 'Authorization: Bearer ..IOsQSrtpSnKTmv70DRwZE9hlh8UkZ7OOu1J2G6N96oE' \
  -H 'Content-Type: application/json' \
  -H 'Billz-Response-Channel: HTTP' \
  --data-raw '
  {
    "ordered_measurement_value": 1,
    "product_id": "aba9d9ad-5d37-4836-81a4-9a60648d087e",
    "retail_price": 110000,
    "supplier_order_id": "e8f19b80-6b2c-41ec-ad0f-0f4fe60aa191",
    "supply_price": 1
}
  '
```

### Изменение неотправленного заказа

Пока заказ не отправлен, его можно отредактировать.

входящая модель точно такая же как и при создании заказа

**пример вызова**

```bash
curl 'https://api-admin.billz.ai/v2/supplier-order/update/:id \
  -X 'PUT' \
  -H 'Authorization: Bearer ..' \
  -H 'Billz-Response-Channel: HTTP' \
  --data-raw '
  {
    "accepting_date": "2024-12-17",
    "settlement_type": "commission",
    "shop_id": "70919136-9481-4c7e-8581-685d750d67a9",
    "supplier_id": "5f4c5af2-3320-4fa4-aed9-65a35e1dab8f",
    "name": "новый заказ 2",
    "comment": ""
}'
```

### Отправка заказа

После того, как заказ собрали, его можно перевести в статус Оформлен

**параметры**

| **поле** | **описание** |
| --- | --- |
| id | идентификатор заказа |

**пример вызова**

```bash
curl 'https://api-admin.billz.ai/v2/supplier-order/change-status' \
  -X 'PATCH' \
  -H 'Authorization: Bearer ..' \
  -H 'Content-Type: application/json' \
  -H 'Billz-Response-Channel: HTTP' \
  --data-raw '
  {
    "id": "e8f19b80-6b2c-41ec-ad0f-0f4fe60aa191"
	}
  '
```

### Список заказов

Метод позволяет получать список заказов в магазине.

```bash
curl -X 'GET' \
  'https://api-admin.billz.ai/v2/supplier-order?limit=10&page=1' \
  -H 'accept: application/json' \
  -H 'Authorization: Bearer ...'
```

- **from_supply_price**	Минимальная закупочная цена

- **from_measurement_value**	Минимальное количество товара
- **to_measurement_value**	Максимальное количество товара
- **limit**	Количество записей на странице
- **page**	Номер страницы
- **shop_ids**	Идентификаторы магазинов
- **start_date**	Начальная дата создания заказа
- **end_date**	Конечная дата создания заказа
- **accept_start_date**	Начальная дата принятия заказа
- **accept_end_date**	Конечная дата принятия заказа
- **status_id**	Идентификатор статуса заказа
- **supplier_ids**	Идентификаторы поставщиков

### Список продуктов

Товары, добавленные к заказу можно получить отдельным списком, указав сам

```bash
curl 'https://api-admin.billz.ai/v2/supplier-order-products/:id?page=1&limit=10&status=added&shop_id=:shop_id' \
  -X 'GET' \
  -H 'authorization: Bearer ..'
```

**Входящие параметры**

- **shop_id**	магазин, в который сделали заказ
- **status**	статус для добавленных товаров “added”
- **limit**	Количество записей на странице
- **page**	Номер страницы

**Исходящая модель**

| **Поле** | **Описание** |
| --- | --- |
| count | Количество элементов |
| product_ids | **Идентификаторы товаров списком** |
| [items.id](http://items.id/) | внутренние ID детали заказа |
| items.supplier_order_id | Идентификатор заказа поставщика |
| items.product_id | Идентификатор товара |
| items.old_retail_price | Старая розничная цена товара |
| items.old_supply_price | Старая закупочная цена товара |
| items.retail_price | Текущая розничная цена товара |
| items.supply_price | Текущая закупочная цена товара |
| items.ordered_measurement_value | **Количество заказанного товара** |
| items.small_left_measurement_value | Оставшееся количество товара |
| [items.product.id](http://items.product.id/) | **Идентификатор товара внутри объекта "product"** |
| [items.product.name](http://items.product.name/) | Название товара |
| items.product.sku | Артикул товара |
| items.product.barcode | Штрихкод товара |
