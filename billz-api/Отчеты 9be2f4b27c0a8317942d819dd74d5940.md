### Сводный

Общая статистика по основным показателям магазина включая статистику продаж, продуктов, клиентов, а также продавцов. Входящие параметры:

- дата с … - по …
- пагинация (page, limit)
- список ид магазинов через запятую
    - ⚡️[Список магазинов](%D0%9C%D0%B5%D1%82%D0%BE%D0%B4%D1%8B%20c242f4b27c0a820d856c81c9179e4051.md)

    - [⚡️Список валют](%D0%9C%D0%B5%D1%82%D0%BE%D0%B4%D1%8B%20c242f4b27c0a820d856c81c9179e4051.md)
- детализация (по дням, месяц…)

## Продажи по товарам

Данный отчет покажет вам какие товары продаются хорошо. Вы сможете посмотреть разбивку по категориям, цветам, размерам и другим характеристикам товаров. Входящие параметры:

| **параметр** | **описание** |
| --- | --- |
| **start_date** | Период продаж с |
| **end_date** | Период продаж до |
| **page** | страница |
| **limit** | кол-во строк на таблицу |
| shop_ids | список магазинов через запятую |
| currency | валюта отчёта |
| detalization | уровень группировки отчёта |
| **detalization_by_position** | детализация по каждой позиции в чеке |
| price_type | фильтр по типу цен |
- Период продаж с … - по …
    - **start_date:** *YYYY-MM-DD*
    - **end_date:** *YYYY-MM-DD* ( не обязателен для 1 дня)
- пагинация (page, limit)
    - **page**: min 1
    - **limit**: max 1000
- список UUID магазинов через запятую
    - ⚡️[Список магазинов](%D0%9C%D0%B5%D1%82%D0%BE%D0%B4%D1%8B%20c242f4b27c0a820d856c81c9179e4051.md)
- валюта, код (UZS, USD, KZT, etc)
    - [⚡️Список валют](%D0%9C%D0%B5%D1%82%D0%BE%D0%B4%D1%8B%20c242f4b27c0a820d856c81c9179e4051.md)
- детализация группировки по периоду
    - [⚡️Детализация (по дням, неделям, …..)](%D0%9C%D0%B5%D1%82%D0%BE%D0%B4%D1%8B%20c242f4b27c0a820d856c81c9179e4051.md)
        - нет значения: без группировки
        - **day**: по дням
        - **week**: по неделям
        - **month**: по месяцам
        - **year**: по годам
- Фильтр типа цены продаж
    - **price_type**:
        - 0 - все цены, по умолчанию
        - 1 - розничные цены
        - 2 - свободные цены
        - 3 - оптовые цены
- Признак группировки цен
    - **detalization_by_position**
        - false — по умолчанию выключена
        - true — группировать по ценам

## **Импорты**

Данный отчет используется, чтобы проанализировать, как продаются поставки, которые были сделаны в определенный период. Входящие параметры:

- дата с … - по …
- пагинация (page, limit)
- **import_type** тип импорта
    - импорты — **import**
    - заказы — **supplier_order**
    - заказы и импорты — не передавать
- детализация (по дням, месяц…)
    - [⚡️Детализация (по дням, неделям, …..)](%D0%9C%D0%B5%D1%82%D0%BE%D0%B4%D1%8B%20c242f4b27c0a820d856c81c9179e4051.md)

## **Продажи по поставщикам**

Отчет покажет как продаются товары в разбивке по поставщикам. Входящие параметры:

    - **limit**: max 1000

    - [⚡️Список поставщиков](%D0%9C%D0%B5%D1%82%D0%BE%D0%B4%D1%8B%20c242f4b27c0a820d856c81c9179e4051.md)

    - [⚡️Список валют](%D0%9C%D0%B5%D1%82%D0%BE%D0%B4%D1%8B%20c242f4b27c0a820d856c81c9179e4051.md)

## **Финансовые транзакции**

Детальный отчет по финансовым траназкциям. 

Входящие параметры:

- дата с … **start_date** - по … **end_date**
- пагинация (**page**, **limit**)
- **shop_ids** — список ид магазинов через запятую
    - ⚡️[Список магазинов](%D0%9C%D0%B5%D1%82%D0%BE%D0%B4%D1%8B%20c242f4b27c0a820d856c81c9179e4051.md)
- **operation** — Типы операций, через запятую
    - 1 — Доход
    - 2 — Расход
    - 3 — Закрытие смены
    - 4 — Перемещение
    - 5 — Конвертация
    - 6 — Инкассация
    - 7 — Открытие кассы

## Перемещения

Детальный отчет по перемещениям товаров. 

Входящие параметры:

- дата с … **start_date** - по … **end_date**
- пагинация (**page**, **limit**)
- **shop_ids** — список ид магазинов через запятую
    - ⚡️[Список магазинов](%D0%9C%D0%B5%D1%82%D0%BE%D0%B4%D1%8B%20c242f4b27c0a820d856c81c9179e4051.md)
- **group_by** — как группировать товары
    - no value — без группировки
    - transfer_name — по названию перемещения
    - product_name — по наименованию товара
    - sku — по артикулу
