# Клиенты

## Создание клиента

Метод позволяет создать клиента.

Endpoint: `POST /v1/client`

Параметры запроса

| Имя | Тип | Описание |
| --- | --- | --- |
| chat_id | string | chat_id клиента в telegram |
| date_of_birth | string | Дата рождения. Формат даты `2022-05-14` |
| first_name | string | Имя клиента |
| last_name | string | Фамилия клиента |
| phone_number | string | Телефон клиента |
| gender | integer | Пол клиента. 
0 - не определен
1 - мужской
2 - женский |

Пример запроса

```bash
curl -X 'POST' \
  'https://api-admin.billz.ai/v1/client' \
  -H 'accept: application/json' \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer ..' \
  -d '{
  "chat_id": "some_chat_id",
  "date_of_birth": "2022-05-14",
  "first_name": "John",
  "gender": 1,
  "last_name": "Doe",
  "phone_number": "+998000000000"
}'
```

Параметры ответа

| Имя | Тип | Описание |
| --- | --- | --- |
| id | string (uuid) | Идентификатор созданного клиента |

Пример ответа

```json
{
	"id": "00000000-0000-0000-0000-000000000000"
}
```

## Получение списка клиентов

Метод позволяет получить краткую информацию по списку клиентов.

Endpoint: `GET /v1/client` 

Параметры запроса

| Имя | Тип | Описание |
| --- | --- | --- |
| chat_id | string | chat_id клиента в telegram |
| limit | integer | Количество элементов на страницу. По умолчанию - 10 |
| page | integer | Номер страницы. По умолчанию - 1 |
| search | string | Строка поиска |
| phone_number | string | Телефон клиента |

Пример запроса

```bash
curl -X 'GET' \
  'https://api-admin.billz.ai/v1/client' \
  -H 'accept: application/json' \
  -H 'Authorization: Bearer ..' \
```

Параметры ответа

| Имя | Тип | Описание |
| --- | --- | --- |
| count | int | Кол-во клиентов в базе |
| clients | array[object] | Список клиентов |
| clients.balance | float | Баланс клиента |

| clients.phone_numbers | []string | Список номеров клиента |
| clients.cards | []string | Список карт клиентов |
| clients.chat_id | string | Telegram chat_id клиента |
| clients.id | string (uuid) | Идентификатор клиента |
| clients.created_at | string | Дата создания клиента |
| clients.last_transaction_date | string | Последняя дата транзакции |

Пример ответа

```json
{
```

## Редактирование клиента

Метод для редактирования данных клиента. Идентификатор клиента должен передаваться в пути запроса.

Endpoint: `PUT /v1/client/{id}`

Параметры запроса

| Имя | Тип | Описание |
| --- | --- | --- |
| date_of_birth | string | Дата рождения. Формат даты `2022-05-14` |
| first_name | string | Имя клиента |
| last_name | string | Фамилия клиента |
| phone_number | array[string] | Номера клиента, перечисленные массивом |
| gender | integer | Пол клиента. 
0 - не определен
1 - мужской
2 - женский |

Пример запроса

```bash
curl -X 'PUT' \
  'https://api-admin.billz.ai/v1/client/123' \
  -H 'accept: application/json' \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer ..' \
  -d '{
  "date_of_birth": "2022-05-14",
  "first_name": "John",
  "gender": 1,
  "last_name": "Doe",
  "phone_number": ["+998000000000"]
}'
```

Параметры ответа

| Имя | Тип | Описание |
| --- | --- | --- |
| message | string | Сообщение об успешном обновлении |

Пример ответа

```json
{
	"message": "Successfully updated"
}
```

## Изменение кешбека

Это ручная операция изменения кешбека. Используйте с осторожностью. 

Метод принимает новое значение кешбека и устанавливает покупателю.

Endpoint: `PATCH /v1/customer/set-balance/{id}`

Параметры запроса

| Имя | Тип | **Описание** |
| --- | --- | --- |
| balance | integer | Новое значение кешбека |

```json
{
    "balance": "943000"
}
```

Пример запроса

```bash
curl 'https://api-admin.billz.ai/v1/customer/set-balance/622c8993-9f82-45ae-a69d-63103d8506d5' \
  -X 'PATCH' \
  -H 'Authorization: Bearer ..' \
  --data-raw '{"balance":943000}'
```

Параметры ответа

| Имя | Тип | **Описание** |
| --- | --- | --- |
| message | string | Идентификатор запроса |

Пример ответа

```json
{
	"message": "00000000-0000-0000-0000-000000000000"
}
```
