# API документация

<aside>
⚠️

Внимание!

Это документация к API системы BILLZ 2.0

Документация к API системы BILLZ 1.0 находится по этой ссылке [https://api.billz.uz/docs/#/](https://api.billz.uz/docs/#/)

BILLZ1 и BILLZ2 не связанные между собой системы

</aside>

Интерфейс BILLZ 2.0 REST API используется для интеграции eCommerce систем с системой BILLZ 2.0 APP. Весь доступ к API осуществляется через HTTPS и доступен с [https://api-admin.billz.ai](https://api-admin.billz.ai). Все данные отправляются и принимаются в формате JSON**.**
