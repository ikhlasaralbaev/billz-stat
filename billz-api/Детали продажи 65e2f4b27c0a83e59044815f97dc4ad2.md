# Детали продажи

При необходимости можно запросить детали одной продажи, например для работы со списком товаров в корзине завершённой транзакции

**Входящие параметры**

передаётся только UUID продажи, обмена или черновика

**Пример запроса**

```bash
curl 'https://api-admin.billz.ai/v2/order/:id' \
  -H 'Authorization: Bearer ...' 
  
```

**Модель ответа**

Ответ включает в себя список продаж, сгруппированных по дате продаже
