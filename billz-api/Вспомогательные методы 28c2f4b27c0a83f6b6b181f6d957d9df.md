### Компания

Информация по текущей компании

```bash
curl -X 'GET' \
  'https://api-admin.billz.ai/[v1/company](https://test-billz-vadimtest2.billz.ai/api/v1/company)' \
  -H 'accept: application/json' \
  -H 'Authorization: Bearer ...'
```

### История курсов валют

История курсов с возможностью фильтрации на дату начала действия курса

*Параметры*

- дата начала действия курса с … - по …
- пагинация (page, limit)

**Пример запроса**

```bash
curl 'https://api-admin.billz.ai/v2/company-currency-rates?limit=5&page=1&currency=&start_date=2020-01-01&end_date=2024-07-26' \
  -H 'authorization: Bearer ..' \
  -H 'accept: application/json'
```

**Пример ответа**
