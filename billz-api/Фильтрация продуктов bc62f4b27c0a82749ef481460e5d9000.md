# Фильтрация продуктов

Продукты в каталоге могут быть отфильтрованы по нескольким параметрам. Часть значений передаются списком, часть единичным значением

| **Параметр** | **Описание** | **Данные** |
| --- | --- | --- |
| **shop_ids** | Магазины | Список ID магазинов |
| **category_ids** | Категории | Список ID категорий |
| **skus** | Артикулы | Список ID артикулов |
| **measurement_unit_ids** | Единицы измерений | Список ID единиц |
| **brand_ids** | Бренды | Список ID брендов |
| **supplier_ids** | Поставщики | Список ID поставщиков |
| **product_field_filters** | Фильтр по аттрибутам или кастомным свойствам | массив объектов-свойств продукта |
| **supply_price_from…supply_price_to** | Цена поставки | Два значения диапазона цены поставки |
| **retail_price_from…retail_price_from** | Цена продажи | Два значения диапазона цены продажи |
| **whole_sale_price_from…whole_sale_price_from** | Оптовая цена | Два значения диапазона оптовой цены |
| **group_variations** | Группировка вариаций | Нужно ли группировать по вариациям (да\нет) |

**Свойство продукта**

Для фильтрации по аттрибутам или кастомным свойства используется один и тот же  параметр **product_field_filters.** В него передаётся массив в 

| **Параметр** | **Описание** |
| --- | --- |
| field_id | ID свойства |
| field_type | Тип свойства: *custom или attribute* (да тут нет опечатки) |
| field_values | Массив ID атрибутов или массив значений кастомных свойств для поиска |

*Атрибут*

```json
  		{
            "field_id": "491e1d8a-ad22-4180-bcc5-0c5699bdf4f9",
            "field_type": "attribute",
            "field_values": [
                "aee6d71d-32ed-4695-8f3b-5516dacf75fe"
            ]
        },
```

Кастомное свойство

```json
{
    "field_id": "3b6db763-eeaf-41c3-af99-48472add4e2e",
    "field_type": "custom",
    "field_values": [
        "18"
    ]
}
```

### Группировка вариаций

Если передается параметр  **"group_variations": true**, то в главном списке будет родительский товар вместо вариаций. А сами вариации находятся в поле **variations.**

**Пример вызова**

```bash
curl --request POST \
  --url https://api-admin.billz.ai/v2/product-search-with-filters \
  --header 'content-type: application/json' \
  --data '{
	"status": "all",
	"order": [
		""
	],
	**"group_variations": true,**
	"product_field_filters": [],
	"field_search_key": "",
	"archived_list": false,
	"brand_ids": [],
	"supplier_ids": [],
	"measurement_unit_ids": [],
	"is_free_price": null,
	"statistics": true,
	"limit": 10,
	"page": 14
}'
```
