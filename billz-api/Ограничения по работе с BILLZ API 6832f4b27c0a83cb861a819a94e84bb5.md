# Ограничения по работе с BILLZ API

# 1. IP и ГЕО ограничения

BILLZ API позволяет подключаться со **всех** IP адресов, без ограничений по странам.

# 2. Rate limit

BILLZ API поддерживает до **2** запросов в секунду с одного IP адреса, но если ресурсы доступны, то лимиты поднимаются.

При достижении порога, клиент получит код ошибки **429.**

# 3. Прямое подключение фото с CDN на сторонние сайты

BILLZ может менять URL адреса фото товаров и гарантировать, что медиа-контент отображается в продуктах BILLZ: Web, Desktop, Mobile.

BILLZ **не разрешает** отображать фото товаров **напрямую** с нашего CDN.

Если у вас есть необходимость показывать медиа контент, то его необходимо поместить на **свой** сервер или CDN и отображать **со своих ресусров**.

# 4. Автоматический блок

Так как в BILLZ работает эвристический анализатор трафика, то некоторые IP адреса могут попадать в блокировку: это происходит при подозрении на  DDOS или бот-трафик.
