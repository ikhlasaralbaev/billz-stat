# Добавление/редактирование кастомного поля товара

Метод позволяет добавлять/редактировать кастомное поле товара. Если id в параметрах уже существует, то произойдет редактирование. В противном случае - добавление.

Endpoint: `PUT /v2/product/{product_id}/patch-props`

Параметры запроса

| Имя | Тип | Описание |
| --- | --- | --- |
| custom_fields | []custom_fields | Список кастомных свойств товара |

```json
{
	"custom_field_id":"00000000-0000-0000-0000-000000000000",
	"custom_field_value":"some value"
}
```

Пример запроса

```bash
curl -X 'PATCH' \
  'https://api-admin.billz.ai/v2/product/:product_id/patch-props' \
  -H 'accept: application/json' \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer {token}' \
  -H 'Billz-Response-Channel: HTTP'
  -d '{
	    "custom_fields": [
		    {
			    "custom_field_id": “{field_id}”, 
			    "custom_field_value": “{field_value}”
			  }
		  ]
}'
```

Параметры ответа

| Имя | Тип | Описание |
| --- | --- | --- |
| session_id | string (uuid) | Идентификатор сессии клиента |
| status_code | number | Статус код HTTP |
| error | object | Объект возвращаемой ошибки |
| error.code | string | Код возвращаемой ошибки. Пустая строка, если ошибки нет |
| error.message | string | Текст возвращаемой ошибки. Пустая строка, если ошибки нет |
| correlation_id | string (uuid) | Идентификатор запроса |
| topic | string | Название топика обработчика |

Пример ответа

```sql
{
    "session_id": "00000000-0000-0000-0000-000000000000",
    "status_code": 200,
    "error": {
        "code": "",
        "message": ""
    },
    "correlation_id": "00000000-0000-0000-0000-000000000000",
    "topic": "some topic name"
}
```
